# 🌱 OptiCrop - Smart Agricultural Production Optimization Engine

## Overview

OptiCrop is an AI-powered crop recommendation system that helps farmers select the most suitable crop based on soil nutrients and environmental conditions.

The application uses a Random Forest Machine Learning model trained on agricultural data to predict the best crop from 22 different crop types.

---

## Features

- AI-powered crop recommendation
- Supports 22 crops
- Interactive web interface
- Crop details and farming tips
- Responsive design
- Prediction confidence score
